# Proof: Exponentiation (x^y)

## Geometric Method

Constructing $x^y$ requires advanced curve fitting, relying heavily on the derivation of the natural logarithm ($\ln$) and Euler's number ($e$).

1. By definition, $x^y = e^{y \cdot \ln(x)}$.
2. Since both multiplication and $e^z$ are constructible operations (proven elsewhere), if we can construct $\ln(x)$, we can construct $x^y$.
3. Utilizing PowerPoint's power regression fit, polynomials up to the 6th degree can be extracted with tight approximations, or $e^x$ can be utilized explicitly by exploiting the exponential curve fit tool.

# PowerPoint Constructibility Framework

A mathematical framework for verifying constructibility proofs within PowerPoint.

## Live Site
Check out the live interactive graph here: [https://jayyswan.github.io/ppt/](https://jayyswan.github.io/ppt/)

## About
This project formalizes the construction of mathematical entities (Points, Segments, Curves) using PowerPoint's native operations. It provides a visual graph of dependencies and proofs for various mathematical constants and shapes.
